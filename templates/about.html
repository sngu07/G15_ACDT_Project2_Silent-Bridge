Project Guide & Manual

Overview

This project is a Sign Language Controlled Robot System utilizing Raspbot and Computer Vision (MediaPipe). The system recognizes hand gestures via webcam and navigates the robot to specific locations on the map.

Commands

Perform the following gestures in front of the camera:

Label Gesture Action Description
TOILET
Go to Toilet Top area of the map.
ROOM 1
+
Go to Room 1 Bottom-center room.
ROOM 2
+
Go to Room 2 Top-center room.
ELEVATOR
Go to Elevator Bottom area (Lobby).
THANK YOU
Return Home Robot returns to base.

Tips

  • Keep your hands 50cm ~ 1m away from the camera.
  • Ensure your hands stay within the camera frame.
  • The robot responds when confidence exceeds 80%.
